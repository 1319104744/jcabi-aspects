Type Changes By

Entire package is more exception safe Fixes 172. yegor

Release 0.7.13 - 7 Mar 2013

Type Changes By
Methods annotated with @Valid are validated for validness of their responses, just like method arguments are validated with the same JSR-303 annotation Fixes 155. yegor
@Loggable can annotate entire class to enable logging of every method in a class (private, protected, and private) Fixes 129. yegor
@Loggable trimming mechanism is configurable with an extra parameter Fixes 156. yegor

Release 0.7.11 - 5 Mar 2013

Type Changes By
Methods annotated with JSR-303 @NotNull are validated for non-nullness of their responses Fixes 132. yegor

Release 0.7.9 - 27 Feb 2013

Type Changes By

Release 0.7.8 - 17 Feb 2013

Type Changes By
@Immutable annotation added to control class immutability in runtime Fixes 133. yegor

Release 0.7.6 - 6 Jan 2013

Type Changes By

Release 0.7.2 - 29 Aug 2012

Type Changes By
